# Muscat 2040: Population & Infrastructure Challenge 🏙️

## Project Overview

This project was developed for the **Rihal Data Analytics Challenge**.\
It provides a **data-driven forecast of Muscat Governorate's population
growth through 2040** and analyzes the resulting impact on essential
infrastructure---specifically **Healthcare and Housing**.

## 📊 Key Findings

-   **Population Range:** Projected to reach **between 1.85M and 2.68M
    by 2040** depending on growth scenarios.
-   **Healthcare Risk:** Current hospital bed capacity is expected to
    reach a **critical breakpoint as early as 2029** under high-growth
    conditions.
-   **Housing Demand:** A potential **shortfall of over 110,000 housing
    units by 2040** if current migration trends accelerate.

------------------------------------------------------------------------

## 🛠️ The Model

The **interactive model** (built with **Python and Streamlit**) allows
users to dynamically adjust key parameters:

-   Annual Population Growth Rate (%)
-   Migration Impact Factor
-   Infrastructure Benchmarks (e.g., hospital beds per 1,000 people)

The dashboard automatically updates the **Gap Analysis** and identifies
the **specific year when infrastructure capacity is exceeded**.

------------------------------------------------------------------------

## 📋 Technical Appendix & Methodology

**Baseline Data** - Sourced from the **National Centre for Statistics
and Information (NCSI), Oman**.

**Projection Model** - Uses **Compound Annual Growth Rate (CAGR)**
calculations to forecast population trends.

**Infrastructure Benchmarks** - Based on **WHO standards and
international urban planning benchmarks**.
